# Login Form With Validation Of Input Field

```
<?php
echo "<pre>";
print_r($_POST);
echo "</pre>";
if(isset($_POST['ulogin']))
{
    extract($_POST);
    if(empty($uemail) and empty($upass) ){
        $msg="Please Fill All Content";
    }
    else if($uemail==''){
        $msg="Please Enter Your Eamil Id";
    }
    else if($upass==''){
        $msg="Please Enter Your Password";
    }
    else{
    $msg="Welcome";    
    }
}
?>
```
